## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- Expo CLI

### Installation

1. Clone the repository:
```bash
git clone https://github.com/helenzhouu/quest.git
cd stem-quest
```

2. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm start
```

5. Run on your device:
```bash
# For iOS
npm run ios

# For Android
npm run android
```
